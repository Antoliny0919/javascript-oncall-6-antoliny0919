# 미션 - 개발자 비상근무

## 기능 목록
  - 사용자 입력
    - [x] 비상근무 배정 월, 시작 요일 입력
    - [x] 평일, 휴일 비상 근무 사원 이름 입력
  - 비상 근무표
    - [] 평일 근무 큐
    - [] 휴일 근무 큐
    - [] 법정공휴일 기록
    - [] 연속2일 근무시 순서 변경
    - [] 근무 기록 내역 기록(x월 y일 z요일 이름)
  - 출력
    - [] 근무 기록 내역 출력

## 예외 목록
  - 비상근무 배정 월, 시작 요일 입력 예외
    - [x] 비상근무 배정 월 입력이 숫자가 아닌 경우
    - [x] 비상근무 배정 월 입력이 1 ~ 12 내가 아닌 경우
    - [x] 비상근무 배정 시작 요일 입력이 (월 ~ 일)이 아닌 경우
  - 평일, 휴일 비상 근무 사원 이름 입력 예외
    - [x] 중복되는 이름이 존재하는 경우
    - [x] 이름이 5자가 초과한 경우
    - [x] 평일, 휴일 근로자(비상 근무자)가 5명 미만인 경우
    - [x] 평일, 휴일 근로자(비상 근무자)가 35명을 초과하는 경우

## 테스트 목록
  - 비상근무 배정 월, 시작 요일 입력 예외 테스트
    - [x] 비상근무 배정 월 입력이 숫자인 경우 테스트
    - [x] 비상근무 배정 월 입력이 1 ~ 12 내가 아닌 경우 테스트
    - [x] 비상근무 배정 시작 요일 입력이 (월 ~ 일)이 아닌 경우 테스트
  - 평일, 휴일 비상 근무 사원 이름 입력 예외
    - [x] 중복되는 이름이 입력된 경우 테스트
    - [x] 5자가 초과한 이름이 존재할 경우 테스트
    - [x] 평일, 휴일 근로자(비상 근무자)가 5명 미안인 경우 테스트
    - [x] 평일, 휴일 근로자(비상 근무자)가 35명을 초과한 경우 테스트
